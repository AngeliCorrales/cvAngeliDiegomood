Proyecto

Desarrollo de una Plataforma Web para Gestión de Inventarios

Fui responsable del diseño, desarrollo e implementación de un sistema de gestión integral para proyectos de software, diseñado para optimizar el flujo de trabajo y la comunicación entre equipos dentro de una empresa de tecnología. El sistema permite el seguimiento de las tareas del proyecto, la asignación de recursos, la gestión de plazos y la generación de informes detallados sobre el progreso de cada fase del proyecto. Este sistema fue implementado como una solución interna para la gestión de múltiples proyectos simultáneos con equipos distribuidos.

Responsabilidad

  • Análisis de Requerimientos: Colaboré con el equipo de operaciones para identificar las necesidades del sistema y los requisitos funcionales.
  • Desarrollo de Software: Utilicé tecnologías como HTML, CSS, JavaScript, Node.js y MySQL para desarrollar tanto el front-end como el back-end de la aplicación.
  • Integración con API's Externas: Implementé integraciones con servicios de proveedores externos para la actualización automática de los inventarios y la gestión de pedidos.
  • Pruebas de Calidad: Realicé pruebas exhaustivas para asegurar la funcionalidad, escalabilidad y seguridad de la aplicación.
  • Despliegue y Mantenimiento: Desplegué la plataforma en un servidor AWS y gestioné su mantenimiento y actualización continua.

Resultados

  • Reducción del 30% en los errores de inventario y tiempos de inactividad del sistema.
  • Aumento del 20% en la eficiencia de la gestión de inventarios gracias a la automatización de procesos.
  • Plataforma utilizada por más de 50 empleados de la empresa.

Este tipo de proyecto demuestra habilidades clave como el análisis de requerimientos, el desarrollo de aplicaciones, la integración de sistemas y el trabajo en equipo, que son muy valoradas en el campo de la ingeniería en sistemas.
